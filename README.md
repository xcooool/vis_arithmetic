# vis_arithmetic Code 
We have four directories.
* embedding _arithmetic:
all baselines for method embedding_arithmetic, including zerocap, imagebind and languagebind.
* evalution:
IRPD dataset evaluation code and visual7w-telling evaluation code
* IRPD_dataset:
IRPD dataset generation pipeline
* sari_rft:
Our models' RFT method, including grpo for two tasks and sft.
